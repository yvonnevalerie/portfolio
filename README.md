# My Dynamic Portfolio

## Overview
My Dynamic Portfolio is a responsive personal portfolio website that showcases my profile, education, skills, projects, and certificates. It was developed using HTML, CSS, JavaScript, and JSON to present my academic background and programming projects in a clean and user-friendly design.

## Features
- Responsive design for desktop and mobile devices
- About Me section
- Education and Skills
- Projects with GitHub and Figma links
- Certificates with a **View Certificate** button
- Contact section with Email, GitHub, and Figma links

## Technologies Used
- HTML5
- CSS3
- JavaScript
- JSON

## Projects Included
- **Suiteness Hotel & Restaurant Website** – A responsive hotel and restaurant website with room reservations, QR code integration, chatbot support, notifications, and customer reviews.
- **Hangman Game** – A Java-based word guessing game with random word selection and score tracking.
- **Hangman Application** – A Java application demonstrating object-oriented programming and problem-solving.
- **Bank Project** – A Java-based Bank Management System implementing account management and banking operations.
